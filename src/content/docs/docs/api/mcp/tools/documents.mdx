### listDocuments

🔍 **Read-only**

List all documents for the organization.

```json
Parameters: {
  "organization_id": "org_xxx",
  "size": 50,
  "cursor": "optional_cursor"
}
```

Supports: Pagination
Returns: Array of documents, `next_cursor` for pagination

```
Example:
AI: "List all policy documents"
    [Uses listDocuments tool]
```
