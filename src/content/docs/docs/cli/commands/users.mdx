---
title: "CLI: Users"
description: Manage users with the Probo CLI
---

The `prb user` commands let you list and view users in your organization.

## List users

```bash
prb user list
```

| Flag                | Default     | Description                                   |
| ------------------- | ----------- | --------------------------------------------- |
| `--org`             | —           | Organization ID                               |
| `--limit`, `-L`     | 30          | Maximum number of results                     |
| `--order-by`        | `FULL_NAME` | Sort field: `FULL_NAME`, `CREATED_AT`, `KIND` |
| `--order-direction` | `DESC`      | Sort direction: `ASC` or `DESC`               |
| `--active`          | —           | Only show users with active contracts         |
| `--json`            | —           | Output as JSON                                |

## View a user

```bash
prb user view USER_ID
```

Displays user profile details including name, email, position, and contract dates.

| Flag             | Description   |
| ---------------- | ------------- |
| `--output`, `-o` | Output format |
