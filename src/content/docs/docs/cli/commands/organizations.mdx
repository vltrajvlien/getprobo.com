title: "CLI: Organizations"
description: List organizations with the Probo CLI
---

The `prb org` commands let you view organizations you have access to.

## List organizations

```bash
prb org list
```

Displays a table with each organization's ID, name, and your role.

| Flag            | Default | Description               |
| --------------- | ------- | ------------------------- |
| `--limit`, `-L` | 30      | Maximum number of results |
| `--json`        | —       | Output as JSON            |
